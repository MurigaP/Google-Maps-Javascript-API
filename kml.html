GMaps.js — KML layers

GMaps.js allows to add KML and GeoRSS layers:

infoWindow = new google.maps.InfoWindow({});
map.loadFromKML({
  url: 'https://www.google.com/maps/d/kml?output=kml&hl=en&t=m&msa=0&ie=UTF8&mid=1jZei-78MB46TKFXkqh0w-STtT3g'',
  suppressInfoWindows: true,
  events: {
    click: function(point){
      infoWindow.setContent(point.featureData.infoWindowHtml);
      infoWindow.setPosition(point.latLng);
      infoWindow.open(map.map);
    }
  }
});

Note: You can learn more about KML and GeoRSS layers here.
