GMaps.js — Polygons

You can add polygons in GMaps.js this way:

  path = [
          [-0.42485801445251414, 36.956020295619965],
          [-0.4207382535755019, 36.97885125875473],
          [-0.4277761770793117, 36.99052423238754],
          [-0.44099372320492347, 36.988979279994965],
          [-0.44305359842727543, 36.967521607875824],
          [-0.43344084253323567, 36.95155709981918]
        ];

map.drawPolygon({
  paths: path,
  strokeColor: '#131540',
  strokeOpacity: 0.6,
  strokeWeight: 6
});

The path of the polygon is defined by an array of array of two (latitude and longitude).

NOTE: Also, you can add a GeoJSON Polygon or MultiPolygon path using useGeoJSON: true.
