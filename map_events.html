GMaps.js — Map events

GMaps.js allows to define map events very easily:

map = new GMaps({
  el: '#map',
  lat: -0.4237207,
  lng: 36.9539251,
  zoom: 16
  click: function(e){
    alert('click');
  },
  dragend: function(e){
    alert('dragend');
  }
});

Note: You can check the list of map events here.
