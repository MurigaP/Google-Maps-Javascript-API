GMaps.js — Polylines

You can add polylines in GMaps.js this way:

  path = [
          [-0.42168236563605355, 36.941460594534874],
           [-0.42228316416044315, 36.947425827383995],
           [-0.4218111081809035, 36.95682428777218],
           [-0.42653166668256465, 36.96609400212765],
           [-0.43142387880142036, 36.95995710790157],
           [-0.43253964603843986, 36.94884203374386],
           [-0.4386334488120785, 36.94249056279659]
         };

map.drawPolyline({
  path: path,
  strokeColor: '#131540',
  strokeOpacity: 0.6,
  strokeWeight: 6
});

The path of the polyline is defined by an array of array of two (latitude and longitude).
