GMaps.js — Basic

Using GMaps.js API is as easy as:

new GMaps({
  el: '#map',
  lat: -0.427787,
  lng: 36.9432071
});

You just define map container ID, latitude and longitude of the map's center and the API does the magic for you.

Note: You also can define zoom, width and height. By default, zoom is 15. Width and height in a CSS class will replace these values.
